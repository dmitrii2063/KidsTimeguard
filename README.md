# KidsTimeguard
приложение для операционной системы Windows 11, предназначенного для контроля времени использования компьютера ребенком. Приложение автоматически отключает компьютер в заданное родителем время.
ПЛАНИРОВЩИК ВЫКЛЮЧЕНИЯ
Версия 1.0 для Windows 11

ОПИСАНИЕ
---------
Приложение автоматически выключает компьютер в заданное время 
и блокирует возможность включения до указанного времени.

ВОЗМОЖНОСТИ
-----------
✓ Автоматическое выключение в заданное время
✓ Блокировка включения в ночное время
✓ Настраиваемое расписание
✓ Работа в системном трее
✓ Защита от отмены выключения

СИСТЕМНЫЕ ТРЕБОВАНИЯ
--------------------
• Windows 11 (сборка 22000 и выше)
• 50 MB свободного места
• Права администратора для установки

УСТАНОВКА
---------
1. Запустите ShutdownScheduler_Setup.exe от имени администратора
2. Следуйте инструкциям мастера установки
3. Настройте время выключения и разблокировки
4. Завершите установку

НАСТРОЙКА
---------
После установки:
1. Откройте приложение из меню Пуск или из трея
2. Настройте время выключения (по умолчанию 22:00)
3. Настройте время разблокировки (по умолчанию 09:00)
4. Нажмите "Сохранить" для применения настроек

ИСПОЛЬЗОВАНИЕ
-------------
• Приложение автоматически запускается при старте системы
• Иконка в трее показывает статус
• Для выключения сейчас: правый клик по иконке -> "Выключить сейчас"
• Для отмены выключения: правый клик -> "Отменить выключение"

БЕЗОПАСНОСТЬ
------------
Приложение использует системные механизмы Windows для блокировки:
• Служба блокировки запускается при старте системы
• При попытке включения в заблокированное время компьютер выключится
• Для полной защиты рекомендуется настроить пароль в BIOS

УСТРАНЕНИЕ ПРОБЛЕМ
------------------
Проблема: Приложение не запускается
Решение: Запустите от имени администратора

Проблема: Компьютер не выключается
Решение: Проверьте права доступа и настройки UAC

Проблема: Можно включить компьютер раньше времени
Решение: Убедитесь, что служба StartupBlocker запущена

УДАЛЕНИЕ
--------
1. Откройте Панель управления -> Программы и компоненты
2. Выберите "Планировщик выключения"
3. Нажмите "Удалить"
